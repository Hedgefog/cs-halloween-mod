# hwn_pumpkin_dispenser

> Drop pumpkins.

### PARAMS
| Param   | Description                                     | Type  | Default Value |
|---------|-------------------------------------------------|-------|---------------|
| delay   | delay before spawn (0.0 - spawn once per round) | float |               |
| impulse | drop velocity.                                  | float |               |

### FLAGS
| Flag | Description            | Default Value |
|------|------------------------|---------------|
| 1    | random drop direction. | false         |
